# File Modes

## r mode : read only
## a mode : append only
## w mode : write only
## r+ or w+ mode : read and write
## a+ mode: read and append
### By default all files are read from and written to via text mode
## For Binary: 
## r+b mode : read in binary
## w+b mode: write in binary
## x mode : create a file if it doesn't exist
